# ✈️ 小航AI助手

郑州航空工业管理学院校园信息查询AI助手

## 项目简介

小航AI助手是一款基于Streamlit和Qwen大模型的校园信息查询应用，旨在为郑州航空工业管理学院的师生提供便捷的校园信息查询服务。

## 功能特性

- 🎓 **身份识别**：支持大一新生、在校老生、教师三种身份选择
- 💬 **智能问答**：基于Qwen2.5-7B模型，根据用户身份提供差异化回答
- 📞 **电话黄页**：包含学校各部门联系方式
- 🔍 **关键词匹配**：自动根据问题关键词匹配相关资料
- 🛡️ **防骗规则**：涉及金钱转账自动提示防骗警告
- 💾 **多轮对话**：支持会话上下文保持
- 📱 **模拟模式**：API服务不可用时自动切换模拟模式

## 技术栈

- **前端框架**: Streamlit
- **AI模型**: Qwen/Qwen2.5-7B-Instruct (SiliconFlow)
- **语言**: Python 3.10+

## 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd xiaohang_helper
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **配置环境变量**
   
   创建 `.env` 文件并添加以下内容：
   ```
   API_URL=https://api.siliconflow.cn/v1/chat/completions
   API_KEY=your-api-key-here
   MODEL_NAME=Qwen/Qwen2.5-7B-Instruct
   ```

4. **运行应用**
   ```bash
   python -m streamlit run src/app.py
   ```

5. **访问应用**
   
   在浏览器中打开 http://localhost:8501

## 使用说明

### 身份选择

首次进入应用需要选择身份：
- **🎓 大一新生**：新入学的同学，了解报到流程、宿舍、学费、军训等信息
- **📚 在校老生**：已在校学习的同学，办理各类事务、查询信息
- **👨‍🏫 教师**：学校教职工，了解办事流程、政策规定等

### 问答功能

1. 选择身份后进入问答页面
2. 可以点击推荐问题快速获取答案
3. 也可以在输入框中输入自定义问题
4. 系统会根据问题关键词自动匹配相关学校资料
5. 回答末尾会标注资料来源

### 电话黄页

点击侧边栏"📞 电话黄页"查看学校各部门联系电话。

## 防幻觉规则

系统严格遵守以下防幻觉规则：
1. 只能根据学校资料回答，资料里没有的明说并建议拨打总值班室电话
2. 严禁编造电话号码、地址、办公时间、学费金额、人名
3. 涉及金钱/转账，无条件提示防骗警告
4. 涉及心理危机，立即提供心理援助方式
5. 不接入学校系统，被问及个人信息时礼貌拒绝
6. 回答末尾标注资料来源
7. 数字必须绝对准确

## 注意事项

- API密钥需从SiliconFlow平台获取
- API服务余额不足时会自动切换到模拟模式
- 模拟模式下使用预设的回答模板
- `.env` 文件已添加到 `.gitignore`，请勿提交到版本库
