# 郑州航空工业管理学院 - 办事流程指南

## 一、在校生办事流程

### 1.1 校园卡办理

#### 补办流程

1. **挂失**：校园卡遗失后，立即拨打挂失电话 0371-61912005 或登录一卡通系统进行挂失
2. **补办**：携带身份证到一卡通服务中心（图书馆一楼）填写补办申请表
3. **缴费**：缴纳补办费用20元
4. **领卡**：3个工作日后凭身份证领取新卡

#### 充值方式

- **自助充值**：通过校园卡自助充值机（食堂、图书馆等地设有）充值
- **在线充值**：登录一卡通系统或关注学校微信公众号充值
- **现金充值**：前往一卡通服务中心窗口充值

### 1.2 学生证办理

#### 新办流程

1. 新生报到时由学院统一办理
2. 填写学生证申请表，粘贴一寸照片
3. 学院审核后提交至学生处制证

#### 补办流程

1. 到学院办公室领取《学生证补办申请表》
2. 填写申请表，辅导员签字确认
3. 携带身份证和申请表到学生处办理
4. 缴纳补办费用10元，7个工作日后领取

### 1.3 在读证明

#### 办理流程

1. 登录学校官网，下载《在读证明申请表》
2. 填写申请表，学院审核盖章
3. 携带申请表到学生处办理
4. 学生处审核后开具在读证明

#### 注意事项

- 在读证明可用于办理贷款、签证等事项
- 证明有效期一般为3个月
- 如需中英文对照证明，需提前说明

### 1.4 成绩证明

#### 办理流程

1. 登录教务系统，打印成绩单
2. 携带成绩单到学院办公室审核盖章
3. 到教务处办理成绩证明

#### 注意事项

- 成绩单需包含所有学期成绩
- 英文成绩单需提前申请翻译
- 办理费用：中文10元/份，英文20元/份

### 1.5 转专业

#### 申请条件

- 大一学年结束后，成绩排名专业前30%
- 无违纪记录
- 符合拟转入专业的要求

#### 申请时间

每年6月，具体时间以教务处通知为准。

#### 申请流程

1. 登录教务系统，提交转专业申请
2. 学院审核，确定推荐名单
3. 参加拟转入专业的考核（笔试或面试）
4. 教务处审批，公示结果
5. 办理学籍变更手续

#### 注意事项

- 转专业只有一次机会，务必慎重考虑
- 跨学科门类转专业需额外审批
- 转专业后需补修新专业的基础课程

### 1.6 休学与复学

#### 休学申请

1. 提交休学申请书，说明原因
2. 提供相关证明材料（如医院诊断证明）
3. 学院审核，签署意见
4. 教务处审批，办理休学手续

#### 复学申请

1. 休学期满前一周提交复学申请
2. 提供相关证明材料
3. 学院审核，教务处审批
4. 办理复学手续，安排课程

#### 注意事项

- 休学最长时间为2年
- 复学后编入下一年级学习

### 1.7 毕业手续

#### 办理流程

1. 核对个人信息和成绩
2. 缴纳欠费，办理退费
3. 提交毕业论文，参加答辩
4. 办理离校手续（图书馆、食堂、宿管等部门签字）
5. 领取毕业证和学位证

#### 注意事项

- 毕业手续通常在每年6月办理
- 如有欠费或未完成学分，将无法按时毕业
- 毕业证遗失不予补办，可申请毕业证明书

## 二、图书馆服务

### 2.1 借阅规则

- **借阅数量**：本科生最多借阅10本，研究生最多借阅20本
- **借阅期限**：30天，可续借一次（续借期限15天）
- **逾期罚款**：每天每册0.1元，超过30天暂停借阅权限

### 2.2 借阅流程

1. 凭校园卡进入图书馆
2. 在检索系统中查询所需图书
3. 根据索书号到相应书架取书
4. 到借书台办理借阅手续

### 2.3 电子资源

图书馆提供丰富的电子资源：

- **中文数据库**：中国知网、万方数据库、维普资讯等
- **外文数据库**：EBSCO、Springer、IEEE等
- **电子图书**：超星电子图书、书生之家等

使用方法：

1. 连接校园网或通过VPN登录
2. 访问图书馆官网，进入电子资源页面
3. 选择所需数据库，使用校园卡号登录

### 2.4 预约服务

如需借阅的图书已被借出，可进行预约：

1. 在检索系统中点击"预约"
2. 图书归还后，图书馆会通知预约人
3. 预约人需在3天内到图书馆领取

## 三、教师常用流程

### 3.1 调课申请

#### 申请条件

- 因特殊原因无法按时授课
- 提前至少3个工作日申请

#### 申请流程

1. 登录教务系统，填写《调课申请表》
2. 选择调课时间和地点
3. 学院教学秘书审核
4. 教务处审批
5. 通知学生调课信息

#### 注意事项

- 每学期调课次数不超过3次
- 调课需安排在同一周内完成
- 需提前通知学生，确保教学不受影响

### 3.2 差旅报销

#### 报销标准

参照河南省财政厅相关规定执行：

- **交通费用**：
  - 省部级及以上：飞机头等舱、高铁一等座
  - 厅局级：飞机经济舱、高铁一等座
  - 其他人员：飞机经济舱、高铁二等座
- **住宿费用**：根据出差城市级别而定，一般不超过300元/天
- **伙食补助**：省内出差50元/天，省外出差80元/天

#### 报销流程

1. 填写《差旅费报销单》
2. 提供发票、行程单、审批单等材料
3. 部门负责人审核签字
4. 财务处审核报销

#### 注意事项

- 报销需在出差结束后1个月内完成
- 发票需真实有效，不得虚假报销
- 严格按照报销标准执行，超支部分自理

### 3.3 科研申报

#### 申报时间

每年3月和9月，具体时间以科研处通知为准。

#### 申报类型

- **国家级项目**：国家自然科学基金、国家社会科学基金等
- **省部级项目**：河南省科技攻关项目、河南省哲学社会科学规划项目等
- **校级项目**：郑州航空工业管理学院科研基金项目

#### 申报流程

1. 登录科研管理系统
2. 在线填写申报书
3. 上传相关材料
4. 学院审核推荐
5. 科研处组织评审
6. 公示立项结果

#### 注意事项

- 申报书需严格按照模板填写
- 确保研究内容新颖，具有学术价值
- 团队成员需具备相应的研究能力

### 3.4 设备报修

#### 报修流程

1. 登录后勤服务系统，填写报修申请
2. 选择报修类型和地点
3. 上传故障照片（可选）
4. 后勤部门安排维修人员
5. 维修完成后确认验收

#### 报修类型

- 办公设备：电脑、打印机、复印机等
- 教学设备：投影仪、音响设备、实验设备等
- 基础设施：水电设施、门窗、家具等

#### 注意事项

- 紧急报修可直接拨打后勤服务电话 0371-61912007
- 维修一般在24小时内响应
- 人为损坏需承担维修费用

### 3.5 会议安排

#### 申请流程

1. 登录学校会议管理系统
2. 填写会议申请表，包括会议主题、时间、地点、参会人员等
3. 部门负责人审核
4. 党政办公室审批
5. 安排会议室和相关服务

#### 注意事项

- 大型会议需提前一周申请
- 会议室使用需遵守相关规定
- 会议结束后及时清理会场

## 四、财务服务

### 4.1 发票开具

#### 开具流程

1. 提供相关合同或证明材料
2. 填写《发票开具申请表》
3. 财务处审核
4. 开具发票

#### 注意事项

- 发票需在业务完成后3个月内开具
- 增值税专用发票需提供完整的开票信息
- 发票一经开出，不得随意更改

### 4.2 经费报销

#### 报销流程

1. 整理报销凭证（发票、收据、合同等）
2. 填写《经费报销单》
3. 部门负责人审核签字
4. 财务处审核报销

#### 注意事项

- 报销凭证需真实有效
- 严格按照经费预算执行
- 报销需在规定时间内完成

### 4.3 工资查询

教师可通过以下方式查询工资：

- **网上查询**：登录财务处官网，进入工资查询系统
- **手机查询**：关注学校微信公众号，绑定工资账户
- **窗口查询**：前往财务处窗口查询

## 五、常见问题解答

### Q：如何查询课程成绩？

A：登录教务系统，进入"成绩查询"页面，选择学年和学期即可查看成绩。

### Q：校园卡丢失怎么办？

A：立即拨打挂失电话 0371-61912005 进行挂失，然后携带身份证到一卡通服务中心补办。

### Q：如何办理缓缴学费？

A：填写《学费缓缴申请表》，提供家庭经济困难证明，经学院审核和财务处批准后可缓缴。

### Q：教师如何申请科研经费？

A：科研项目立项后，根据项目批准书到财务处办理经费入账手续，然后按照经费预算使用。

### Q：如何预约会议室？

A：登录学校会议管理系统，填写会议申请表，经审批后即可预约会议室。

### Q：报销需要哪些材料？

A：报销需提供发票、收据、合同、审批单等材料，具体要求根据报销类型而定。

### Q：如何办理出国手续？

A：教师出国（境）需提前3个月提交申请，填写《因公出国（境）申请表》，经学院和学校审批后办理相关手续。

### Q：如何申请调课？

A：登录教务系统，填写《调课申请表》，经学院教学秘书和教务处审批后即可调课。
