# 郑州航空工业管理学院 - 应急防骗指南

## 一、校园110

### 1.1 校园110简介

校园110是学校设立的安全保卫应急响应系统，旨在维护校园安全秩序，保护师生人身财产安全。校园110实行24小时值班制度，全天候为师生提供安全服务。

### 1.2 报警方式

#### 方式一：电话报警

- **校园110专线**：0371-61911110（24小时）
- **保卫处电话**：0371-61912030

#### 方式二：短信报警

编辑短信发送至 138xxxx1110，内容包括：
- 报警人姓名、学号/工号
- 事发地点
- 事件描述
- 联系方式

#### 方式三：APP报警

下载"郑航平安"APP，通过APP内置的一键报警功能报警。

#### 方式四：现场报警

前往保卫处值班室直接报警，或向校园内的保安人员求助。

### 1.3 报警注意事项

1. **保持冷静**：报警时保持冷静，清晰说明情况
2. **提供准确信息**：包括事发地点、时间、事件性质、人员情况等
3. **保护现场**：在警方到达前，尽量保护好现场，不要移动现场物品
4. **配合调查**：积极配合保卫处和公安机关的调查工作

### 1.4 出警流程

1. **接警**：校园110指挥中心接到报警后，立即记录相关信息
2. **派警**：根据报警内容，派遣就近的巡逻人员前往现场
3. **处置**：巡逻人员到达现场后，进行初步处置
4. **移交**：如涉及刑事案件，移交公安机关处理
5. **反馈**：处理完毕后，向报警人反馈处理结果

### 1.5 巡逻制度

校园110实行24小时巡逻制度：

- **巡逻时间**：全天候不间断
- **巡逻范围**：校园内所有区域，包括教学楼、宿舍楼、图书馆、操场等
- **巡逻方式**：步行巡逻、自行车巡逻、机动车巡逻相结合
- **巡逻重点**：夜间、节假日、人员密集区域

## 二、常见安全隐患与防范

### 2.1 消防安全

#### 火灾隐患

- 宿舍内使用大功率电器（如电热毯、电磁炉、电暖器等）
- 私拉乱接电线
- 吸烟后未熄灭烟头
- 存放易燃物品

#### 防范措施

1. 严禁在宿舍内使用大功率电器
2. 定期检查电线、插座，发现破损及时更换
3. 不在宿舍内吸烟
4. 不存放易燃、易爆物品
5. 熟悉宿舍和教学楼的消防通道和安全出口位置
6. 掌握灭火器的使用方法

#### 火灾应急处理

1. **立即报警**：拨打校园110（0371-61911110）和119
2. **疏散逃生**：按照消防通道指示，迅速撤离到安全区域
3. **灭火**：如火势较小，可使用灭火器进行灭火
4. **协助救援**：配合消防人员进行救援

### 2.2 交通安全

#### 交通安全隐患

- 校园内车速过快
- 行人不遵守交通规则
- 电动车违规行驶
- 停车混乱

#### 防范措施

1. 校园内限速20公里/小时
2. 行人走人行道，过马路注意来往车辆
3. 电动车遵守交通规则，不逆行、不载人
4. 车辆停放在指定区域

#### 交通事故处理

1. **立即停车**：发生交通事故后，立即停车，开启警示灯
2. **保护现场**：设置警示标志，保护好事故现场
3. **救助伤员**：如有人员受伤，立即拨打120和校园110
4. **报警**：拨打122和校园110，等待交警和保卫人员处理

### 2.3 人身安全

#### 人身安全隐患

- 夜间单独出行
- 进入陌生区域
- 与陌生人交往
- 酗酒滋事

#### 防范措施

1. 夜间尽量结伴出行，避免单独前往偏僻区域
2. 不随意进入陌生场所
3. 不轻易相信陌生人，不接受陌生人的馈赠
4. 适量饮酒，避免酒后滋事

#### 紧急情况处理

1. **遭遇袭击**：立即拨打校园110和110，大声呼救，尽量保护自己
2. **失踪人员**：发现人员失踪，立即向保卫处报告，并提供相关信息
3. **意外受伤**：立即拨打120和校园110，进行紧急救助

### 2.4 财产安全

#### 财产安全隐患

- 宿舍门锁不牢
- 贵重物品随意放置
- 校园卡丢失未及时挂失
- 网络诈骗

#### 防范措施

1. 离开宿舍时锁好门窗
2. 贵重物品妥善保管，随身携带或存入保险柜
3. 校园卡丢失后立即挂失
4. 提高警惕，防范各类诈骗

## 三、防骗指南

### 3.1 常见诈骗类型

#### 类型一：电信诈骗

**诈骗方式**：

- 冒充公检法人员，谎称涉嫌犯罪，要求转账
- 冒充学校老师、辅导员，要求缴纳费用
- 冒充快递员，谎称包裹有问题，要求提供个人信息
- 发送中奖信息，要求缴纳手续费
- 发送虚假招聘信息，要求缴纳押金

**防范要点**：

1. 公检法机关不会通过电话办案，不会要求转账
2. 学校老师不会通过电话要求缴纳费用，如有疑问直接联系学院
3. 快递问题直接联系快递公司官方客服
4. 中奖信息需要先交钱的都是诈骗
5. 招聘信息需要缴纳押金的都是诈骗

#### 类型二：网络诈骗

**诈骗方式**：

- 虚假购物网站，骗取货款
- 网络兼职刷单，骗取佣金
- 虚假投资理财，骗取本金
- 冒充QQ、微信好友，骗取钱财

**防范要点**：

1. 购物选择正规网站，注意核对网址
2. 网络兼职刷单是违法行为，且多为诈骗
3. 高收益、低风险的投资项目多为诈骗
4. 好友借钱需通过电话或视频确认身份

#### 类型三：校园诈骗

**诈骗方式**：

- 上门推销假冒伪劣产品
- 冒充学生组织收取费用
- 虚假培训、考证信息
- 校园贷

**防范要点**：

1. 不轻易购买上门推销的产品
2. 学生组织收费会有正式通知和票据
3. 培训、考证选择正规机构
4. 远离校园贷，如需贷款通过正规渠道

#### 类型四：求职诈骗

**诈骗方式**：

- 虚假招聘信息，骗取报名费
- 传销组织，诱骗加入
- 兼职中介，骗取中介费

**防范要点**：

1. 招聘单位收取报名费、培训费的都是诈骗
2. 警惕"高薪、轻松、无门槛"的招聘信息
3. 兼职中介收取高额费用的都是诈骗

### 3.2 防骗技巧

1. **提高警惕**：对陌生电话、短信、信息保持警惕
2. **核实信息**：涉及金钱、个人信息的，务必多方核实
3. **保护个人信息**：不随意泄露身份证号、银行卡号、密码等信息
4. **不贪小便宜**：天上不会掉馅饼，警惕中奖、返利等信息
5. **及时报警**：发现被骗，立即拨打校园110和110

### 3.3 被骗后的处理

1. **保持冷静**：不要惊慌，立即停止与骗子的联系
2. **保留证据**：保存好聊天记录、转账记录、通话录音等证据
3. **立即报警**：拨打校园110（0371-61911110）和110
4. **联系银行**：如涉及银行卡转账，立即联系银行冻结账户
5. **通知家人**：及时告知家人，防止二次诈骗

## 四、心理援助

### 4.1 心理健康教育中心简介

学校心理健康教育中心是为师生提供心理健康服务的专业机构，旨在帮助师生解决心理问题，促进心理健康发展。

### 4.2 服务内容

#### 心理咨询

- **个体咨询**：一对一心理咨询，解决个人心理问题
- **团体咨询**：针对特定主题的团体心理辅导
- **危机干预**：对处于心理危机状态的学生进行紧急干预

#### 心理健康教育

- **心理健康讲座**：定期举办心理健康知识讲座
- **心理健康宣传**：通过多种形式宣传心理健康知识
- **心理健康活动**：组织心理健康相关的校园活动

#### 心理测评

- **新生心理测评**：每年对新生进行心理健康测评
- **日常心理测评**：为有需要的学生提供心理测评服务

### 4.3 预约方式

#### 方式一：电话预约

拨打心理健康教育中心电话：0371-61912023

#### 方式二：现场预约

前往心理健康教育中心（学生活动中心3楼）预约

#### 方式三：在线预约

登录学校心理健康教育中心官网或微信公众号进行预约

### 4.4 咨询流程

1. **预约**：通过电话、现场或在线方式预约咨询时间
2. **初诊**：首次咨询时，咨询师了解基本情况，建立咨询关系
3. **咨询**：根据问题制定咨询方案，进行多次咨询
4. **评估**：咨询结束时，对咨询效果进行评估

### 4.5 服务时间

- **工作日**：8:00-12:00，14:00-18:00
- **周末**：9:00-12:00，14:00-17:00
- **紧急情况**：24小时服务

### 4.6 常见心理问题

#### 适应问题

- **新生适应**：新环境、新学习方式的适应
- **学业适应**：学习压力、考试焦虑
- **人际关系适应**：与同学、老师的相处

#### 情绪问题

- **焦虑**：过度担忧、紧张不安
- **抑郁**：情绪低落、兴趣减退
- **愤怒**：情绪失控、易怒

#### 压力问题

- **学业压力**：课程难度、考试压力
- **就业压力**：找工作的压力
- **经济压力**：家庭经济困难

#### 人际关系问题

- **宿舍关系**：与室友的矛盾
- **恋爱关系**：恋爱中的困惑
- **社交恐惧**：害怕与人交往

### 4.7 心理危机识别

#### 心理危机信号

1. **情绪异常**：突然情绪低落、烦躁、易怒
2. **行为异常**：行为怪异、孤僻、不愿与人交往
3. **言语异常**：说一些消极、绝望的话
4. **生理异常**：失眠、食欲不振、体重明显变化
5. **自杀倾向**：谈论自杀、准备自杀工具

#### 发现心理危机后的处理

1. **保持冷静**：不要惊慌，保持镇定
2. **倾听陪伴**：耐心倾听，给予陪伴和支持
3. **及时报告**：立即向辅导员、学院或心理健康教育中心报告
4. **避免刺激**：不要刺激当事人，不要评判
5. **保护安全**：确保当事人的人身安全

## 五、应急预案

### 5.1 突发公共卫生事件

#### 应急预案

1. **发现报告**：发现疑似病例，立即向校医院和学院报告
2. **隔离观察**：对疑似病例进行隔离观察
3. **医疗救治**：联系医院进行救治
4. **环境消毒**：对相关区域进行消毒
5. **信息发布**：及时发布相关信息，避免恐慌

### 5.2 自然灾害

#### 应急预案

1. **预警信息**：及时发布自然灾害预警信息
2. **疏散转移**：组织师生疏散到安全区域
3. **应急救援**：开展应急救援工作
4. **灾后重建**：进行灾后重建和恢复工作

### 5.3 群体性事件

#### 应急预案

1. **现场处置**：迅速到达现场，控制局势
2. **疏导人群**：疏散围观人群，避免事态扩大
3. **调查处理**：调查事件原因，依法处理
4. **信息发布**：及时发布相关信息，澄清事实
