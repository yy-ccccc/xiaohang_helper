# 04 应急防骗、校园求助渠道
## 校园紧急联系
### 校园110安保报警
1. 校内紧急求助拨打校园110；
2. 适用场景：财物失窃、冲突纠纷、人身安全隐患、外来可疑人员；
3. 24小时值守，龙子湖校区、大学路校区均可统一呼叫；
4. 报警时说明：校区、楼栋、具体位置、事件简要情况。

### 校医院急诊
1. 门诊服务时间：工作日8:00–17:30；
2. 夜间突发伤病先拨打校园110协调应急处置；
3. 重症情况由校医院协助联系120校外急救。

## 反诈防骗指南
### 常见校园诈骗类型
1. 刷单返利诈骗：不存在正规校园刷单任务，先小额返利再大额收割；
2. 冒充客服退款：官方客服不会通过微信、QQ私聊索要银行卡、验证码；
3. 校园贷、分期贷款：校内不允许学生开展非官方网贷，切勿私下借贷；
4. 冒充导师、辅导员QQ/微信借钱：务必电话当面核实身份。

### 被骗之后处置流程
1. 第一时间保存聊天记录、转账凭证、对方账号；
2. 联系学校保卫处反诈专员登记备案；
3. 金额较大直接拨打110报警；
4. 及时告知辅导员。

## 学生心理援助
### 心理咨询预约
1. 心理健康中心面向全体在校生免费开放咨询；
2. 预约渠道：企业微信心理预约平台；
3. 咨询地点：两校区心理健康服务室；
4. 服务范围：学业压力、人际关系、情绪焦虑、情感困扰等。

### 危机紧急干预
1. 存在自伤、极端想法，可直接联系辅导员+心理中心24小时应急热线；
2. 所有咨询内容遵守保密原则，仅在涉及生命安全时启动干预流程。
